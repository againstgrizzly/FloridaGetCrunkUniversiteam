Presentation

We are Florida Get Crunk Universiteam. Our main focus is to give prospective hobbyist to learn how to create their own circuits. Giving them an easy to use simulator which allows you to drag and drop circuit elements and tweak simple environmental settings. Once the circuit is constructed by the user they would be able to start the simulation and interact with any input devices. Thus allowing the user to inspect each part of the circuit in real time, offering a unique and interactive way to build circuits.
