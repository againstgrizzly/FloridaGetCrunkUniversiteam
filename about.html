We are Florida Get Crunk Universiteam.

Our main focus is to give prospective hobbyist to learn how to build their own electrical circuits. Offering an easy to use simulator which allows you to drag and drop circuit components and to configure simple environmental and property settings. Once the circuit is constructed, the user will be able to start the simulation and interact with any input devices. Thus allowing the user to inspect each part of the circuit in real time, offering a unique and interactive way to build circuits.
